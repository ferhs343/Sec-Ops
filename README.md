# ¿Qué es?

Sec-Ops, la herramienta ideal para detectar ataques básicos en un archivo PCAP, acelerando el proceso de análisis y llegar a una conclusión en segundos, así mismo, Sec-Ops fue desarrollado para automatizar procesos de ataques a redes LAN.
